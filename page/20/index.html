July 11, 2010

The final mobile port (for now)

I actually finished on Sunday, but the final mobile port of Smiles is now sent. The port is for Samsung’s Bada mobile platform. I was pleasantly surprised how straight forward it was to get things up and running there. The system API is very much like a C++ version of Apple’s ObjC iPhoneOS API. The whole endeavor took no more than 4 days worth of work. So done! I will continue to support the existing plethora of ports as needed, but now it’s time to switch gears, and get going on the new project!...
